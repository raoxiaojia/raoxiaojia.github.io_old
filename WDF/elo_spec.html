The score function is piecewise-linear, with the following turning points for the losing player:

Match Score Converted Score
0 0
900 0
950 0.05
980 0.15
999 0.25

The converted score for the winning player is 1 minus that of the losing player.

The K-factor is chosen to be 30 for the first 20 matches, and 15 afterwards.

This ELO Rating is updated daily.
