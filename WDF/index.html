About this website

This is an unofficial website of the game dynamite by Softwire.

The aim of this site is to offer much more functions than the official website could, as the developers of the official website are full-time employees hence have a lot of others things to deal with.

ELO Rating

This is an unofficial ELO rating (there is no official version currently) of bots on the official website. Only matches that finished normally (by score) are taken into account.

Better Tournament

The tournament on the original website is very unreliable and inconvenient to use -- for example, there is no way to stop a running tournament; bots are likely to throw errors due to inconsistent time limit and memory limit; format of the tournament is limited (currently only a full round-robin tournament is possible).

This feature is under development.
