Turing Machine

Rules

We are proud to release our next-generation product -- the Turing Machine. To better advertise it, we provide an online sample here and hope to sell some for profit.[1]

The production team is too lazy to provide a comprehensive version of user manual, but they mentioned briefly that the machine takes instructions of the format 'CS,TS,WS,NS,DR', with each token representing:

CS (Current State): a non-negative integer, the current state of the machine;

TS (Tape Symbol): the symbol on the cell of tape that the head of machine is on. Curretly only supporting single charcaters;

WS (Write Symbol): the symbol that the head is going to print, currently only supporting single characters;

NS (Next State): the next state that the state register will be set as. Must be a non-negative integer as well;

DR (Direction): the direction that the head will move next. There are three possible values for it: 'L','R' and '=', which should be self-explanatory.

[1] Note: Due to technical restrictions, we can now only produce a Turing Machine without its tape, its head and its state register. Price starts at $0. Currently 0 in stock.
